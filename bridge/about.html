What Is BerelzBridge

BerelzBridge is an MT5 indicator that exports live market data to a JSON file on your computer. It runs continuously in the background — no EA conflicts, no DLLs, no internet connection required. Your application reads the file at any interval.

It is not a signal generator. It is a data bridge. What you build on top of it is entirely up to you.

What It Exports
BerelzBridge Pro
  • Bid / Ask
  • Spread in points
  • Daily high / low / open
  • Tick volume
  • OHLCV bars × 9 timeframes (up to 500 each): M1, M5, M15, M30, H1, H4, D1, W1, MN1
  • Account: balance, equity, margin, free_margin, profit, leverage, currency
  • Broker: company name, server name
BerelzBridge Free
  • Bid / Ask
  • OHLCV bars × 1 timeframe (50 bars fixed)
